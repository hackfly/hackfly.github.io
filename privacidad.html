Política de Privacidad –

    Run Intervals no recopila, almacena ni rastrea información sobre usted.

    Big Seconds utiliza Google AdMob para mostrar un banner publicitario,
    y este servicio puede recopilar cierta información, incluyendo:

    • Dirección IP, que puede utilizarse para estimar la ubicación general de un dispositivo.
    • Registros de fallos no relacionados con el usuario, que pueden utilizarse para diagnosticar problemas y mejorar el SDK. La información de diagnóstico también puede utilizarse con fines publicitarios y analíticos.
    • Datos de rendimiento asociados al usuario, como el tiempo de inicio de la aplicación, la tasa de bloqueo o el consumo de energía, que pueden utilizarse para evaluar el comportamiento del usuario, comprender la eficacia de las funciones existentes del producto y planificar nuevas funciones. Los datos de rendimiento también pueden utilizarse para mostrar anuncios, incluyendo compartirlos con otras entidades que los muestran.
    • Un ID de dispositivo, como el identificador de publicidad del dispositivo u otros identificadores de dispositivo vinculados a la aplicación, que puede utilizarse para fines publicitarios y analíticos de terceros.
    • Los datos publicitarios, como los anuncios que ha visto el usuario, pueden utilizarse para impulsar las funciones analíticas y publicitarias.
    • Otras interacciones del usuario con el producto, como los toques de inicio de la aplicación y la información de interacción, pueden utilizarse para mejorar el rendimiento de la publicidad.
